Sobre

Iniciei na programação em 2021, criando minhas primeiras páginas com HTML e CSS.

Após um tempo comecei com JavaScript e à partir daí as aplicações começaram a se desenvolver.

Com o tempo passando e o código evoluindo, ficando mais limpo e melhor estruturado, fui aprendendo novas tecnologias como pre processadores CSS e frameworks como o React pra criação de SPAs.

Minha atuação principal é no front-end, porém, pra suprir algumas necessidades e também pra me desenvolver na linguagem comecei a criar APIs com NodeJs conectando a um banco de dados MySQL.

Também busco me desenvolver em estruturas de dados e algoritmos e apesar do uso de frameworks, sempre busco solidificar os conhecimentos em JavaScript vanilla.
